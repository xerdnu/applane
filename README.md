# Applane — Feedback & Support

The place to **request features, ask questions, and report bugs** for
[Applane](https://applane.net) — the workspace for App Store Connect & Google Play metadata.

## Where to go

- 💡 **Idea or feature request** → [Discussions → Ideas](../../discussions/categories/ideas) — and upvote the ones you want.
- ❓ **Questions and help** → [Discussions → Q&A](../../discussions/categories/q-a)
- 🐛 **A bug** → [Open an issue](../../issues/new/choose)

## ⚠️ Please don't post private info here — this repo is public

Never paste:
- Store credentials — service-account JSON, `.p8` keys, Issuer/Key IDs
- Screenshots showing your account, keys, or customer data

For anything tied to your account, use the private channels below.

## 📧 Private support

- **Account, billing, or connection problems** → **support@netblast.se**
- **Security vulnerability** → **security@netblast.se** (or GitHub's [private reporting](../../security/advisories/new)) — please **don't**open a public issue for security.

We usually reply within a day.
